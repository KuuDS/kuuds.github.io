## 新建对象
1. 直接创建实例
    ```javascript
    //实例化后添加属性
    car=new Object();
    car.type="limousine";
    car.brand="BMW";
    
    //使用对象literals
    car=[type:"limousine",brand:"BMW"];
    ```
2. 使用构造器
    ```javascript
    function Car(type,brand){
        //对象属性
        this.type=type;
        this.brand=brand;

        //对象方法
        function setType(type){
            this.type=type;
        }
    }
    
    //实例化对象
    var newCar=new Car("PJ","BMW");
    ```
3. `for ... in ...`可遍历对象中的属性
    ```javascript
    var person={fname:"Bill",lname:"Gates",age:56};
    for (x in person){
        txt=txt + person[x];
    }
    ```
    *该段代码引用自[w3school.com.cn][1]*

## Number对象

2. 数字`0`起头表示八进制，数字`0x`起头表示十六进制
